# 1、注册OpenAI账号

首先，你应该注册一个OpenAI账号，然后使用注册的OpenAI账号，去登录ChatGPT即可，为什么要OpenAI账号呢？因为OpenAI是ChatGPT的母公司，就像你注册一个QQ号或者微信号，去登陆王者荣耀，是一个道理。

要注册OpenAI账号，你要先拥有一个国外的邮箱，例如谷歌邮箱等，但是，请注意，不能使用和微软相关的邮箱(如@outlook.com、@hotmail.com、@live.cn)。

注意

注册OpenAI账号使用的邮箱不能是与微软相关的邮箱如@outlook.com、@hotmail.com、@live.cn，因为微软现在是OpenAI的股东，会把咱国人的个人信息给到OpenAI，而OpenAI是不允许国人使用的。

# 1、科学上网是什么？怎么科学上网？

科学上网指的是访问国外的网络，一般需要通过一定的工具才可以实现，网上有一些免费的工具，但是网络可能不太稳定，还是那句话，好用的东西都是需要付费的，推荐一款GHelper，直接在浏览器添加扩展程序即可。

# 2、没有国外邮箱怎么办？

当你学会科学上网以后，申请国外的邮箱如谷歌邮箱等就不是一件难事，还有一些其他的国外邮箱，如proton (opens new window)。 如果想直接升级到ChatGPT 4.0的话，也可以先在WildCard (opens new window)平台申请虚拟卡后，平台会提供一键注册海外邮箱的服务。

# 3、虚拟信用卡可靠吗？安全吗？

虚拟信用卡也是visa官方发布的信用卡，只是没有实物，其他的基本和实物信用卡无异，可以放心申请使用。
